Claude Token Monitor

A lightweight macOS menu bar app for monitoring Claude Code token usage in real time.

Download for macOS View on GitHub
Screenshot placeholder

Features

📊

Real-time Token Tracking

Monitor input, output, cache read/write tokens with live updates from your active Claude Code sessions.

🔥

Activity Heatmap

GitHub-style calendar heatmap showing your daily token usage patterns at a glance.

📈

Daily Trend Chart

Bar chart visualization of token consumption over 7, 14, or 30 day periods.

📁

Session Breakdown

Expandable list of top sessions with detailed token, cost, and model information.

🎯

Context Usage Indicator

Visual progress bar showing how much of the context window you've consumed.

🎨

Dynamic Tray Icon

Menu bar icon changes based on context usage level — low, medium, or high pressure.

🧊

Liquid Glass Theme

macOS vibrancy-backed translucent theme alongside the default dark theme.

⚡

Power Efficient

Visibility-based polling, cached data, and optimized rendering for minimal battery impact.

🔒

Local & Private

All data stays on your machine. No telemetry, no network requests, no accounts required.
